Practica GIT

Comandos principales:

  • git init: Inicializa un repositorio
  • git add nombre_archivo: Agrega un archivo al area de preparado
  • git commit -m "mensaje": crea una version de la aplicacion
  • git push: guarda en el repositorio remoto (GitHub)
  • git status: verifica el estado de los archivos
  • git log: muestra las versiones guardadas.

Branching

  • git checkout nombre_branch: navega entre ramas
  • git branch: muestra las ramas disponibles
  • git branch nueva_branch: crea una nueva rama
  • git checkout version : navega entre versiones de commit
  • git merge branch_name: combina las ramas
